# Portfolio Project Setup — 100Hires

## Overview

This repository documents the first step of my 100Hires portfolio project: setting up the development environment and tools required to complete the assignment.

---

## Tools Installed

| Tool | Purpose |
|------|---------|
| [Cursor IDE](https://cursor.com/) | AI-powered code editor built on VS Code |
| Claude Code (Cursor Extension) | AI coding assistant by Anthropic, integrated inside Cursor |
| Codex (Cursor Extension) | AI code completion and generation tool |
| Git + GitHub | Version control and public repository hosting |

---

## Steps Completed

1. **Installed Cursor IDE**
   - Downloaded the installer from [cursor.com](https://cursor.com/)
   - Ran the installer and launched the application

2. **Added the Claude Code extension**
   - Opened the Extensions panel in Cursor (`Ctrl+Shift+X` / `Cmd+Shift+X`)
   - Searched for "Claude Code" and installed it
   - Logged in with my Anthropic account

3. **Added the Codex extension**
   - Searched for "Codex" in the same Extensions panel
   - Installed and logged in

4. **Created a GitHub account and public repository**
   - Signed up at [github.com](https://github.com/)
   - Created a new public repository named `For-100Hires`

5. **Opened the repository in Cursor**
   - Cloned the repository locally using the terminal
   - Opened the folder in Cursor via `File → Open Folder`

6. **Created this README.md**
   - Documented all tools, steps, and notes in this file

7. **Committed and pushed to GitHub**
   - Used Git to stage, commit, and push the file to the `main` branch
